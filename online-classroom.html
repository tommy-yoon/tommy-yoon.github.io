Features

  • This software is a web application that allows students attend online-based courses with vide/text chat, white board and other functionalities.
  • All resources for online-classroom are integrated into one application so no need to use several applications(Zoom, Discord, Slack and etc.) for online classes.
  • Timetable is the core part of the software and students not only can check their timetable and their courses but also can attend them directly.
  • Many tasks are automated so that teachers and students can focus on learning.

UI/UX design

  • I designed UI/UX of the software using Figma
  • Figma is one of the best tools to design UI/UX of software.

CI/CD

  • Using GitHub Action, I developed CI/CD pipeline for the software.
  • GitHub Action is good enough for small software but for large software, it can be replaced with other CI/CD such as Jenkins or Travis CI.
  • CI/CD is the process of testing the software before releasing it to the public.
  • The dev pipeline is created for development and testing.
  • The prod pipeline is created for demonstration and user testing.
  • All the configuration for CI/CD is done in a text file using YAML language(link)

Agile

  • I am following Agile methodologies to develop the software using Trello.
  • GitHub project was used for other school projects but Trello provides more features such as board, card, checklist, and so on.
  • Trello is a powerful tool for team to organize their work and communicate with each other.
  • For this project, I integrated Trello and GitHub Action so that all the commits and push requests are updated on Trello cards(link).
