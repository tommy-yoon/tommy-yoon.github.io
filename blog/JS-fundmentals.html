Explain control flow and loops using an example process from everyday life, for example 'waking

up' or 'brushing your teeth'. (But not those ones).

Control flow is the order of activities we do everyday. We get up and then tidy up our bed but we don't tidy up our bed and then get up. We simply cannot do that. We turn our TV on then watch the TV show on the TV. We cannot watch anything before turning the TV on. Likwise, we need an order of activities to achieve our daily goals even if it is a simple goal. That is control flow. Also, that control flow can include repetitive activities. For an example, brushing teeth is looping of strokes for a certain amount of time. Another type of control flow is branching. We need to decide what to do almost every moment. For an instance, we choose to order an double burger when we are starving but a Junior Whopper when we are not that hungry. In JavaScript we implement it with If statements. Using it, we can choose differnt control flows.

Describe what the DOM is and an example of how you might interact with it.

DOM is how HTML is structured. It looks like a big tree and each element eventually connected to the root element. That structure enables us to navigate from the root element to any child element we want to access. So that we can manipulate any element on HTML using JavaScript. For an example we can access a first H1 tag in a HTML page using the below statement.


document.getElementsByTag('h1')

Explain the difference between accessing data from arrays and objects.

Arrays - using index. e.g. myArray[1]
Objects - using property name. e.g. myObject.propertyName or myObject["propertyName"]
